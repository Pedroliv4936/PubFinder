# PUBFinder
## Este projeto tem como objetivo apresentar aos utilizadores os bares, preços e opçoes disponiveis(ementa) na região selecionada.
A aplicação terá uma base de dados de preços preenchida pelos próprios utilizadores. Esta base de dados poderá ser filtrada e
ordenada por preços, proximidade, tipo de bebida, etc. conforme desejado.
Achamos que esta aplicação irá ser útil para pessoas que planeiam sair a noite e pretendem saber o preço de cada um dos bares.
Cenarios reais para a aplicação podem ser:
Principal - O utilizador quer sair com os amigos e não quer gastar muito dinheiro, liga a aplicação e é lhe apresentado a localização dos bares e o preço das bebidas destes.

Secundário - O utilizador quer ir sair para uma determinada zona em que ele não se encontra e saber qual dos bares ele pode gastar menos dinheiro. A aplicação mostra os bares e os precos nessa determinada zona e mostra também a avaliação que outros clientes deram a esse bar.
